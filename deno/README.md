## Deno Bowling Game Kata

### Description

The kata is inspired by the [Bowling Kata](https://codingdojo.org/kata/Bowling/)

### Prerequisite

- [Install deno](https://docs.deno.com/runtime/manual/getting_started/installation)
- Install dependencies

  ```shell
  deno task install
  ```

### Usage

Run the kata tests :

```shell
vr test
```
